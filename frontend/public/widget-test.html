🧪 Widget Test - MutualyF

Página de prueba para verificar que el widget embed funciona correctamente. Pegá tu widget_token (lo generás desde Admin → Configuración) y el widget aparece abajo a la derecha.

1. Pegá tu widget token

¿No tenés token? Logueate en /admin → Configuración → Widget Token → "Generar".

2. Probá la pregunta que devuelve una URL

Una vez cargado el widget, hacé click en la burbuja de chat y preguntá:

¿Cuál es la página web para acceder a Mi MutuaLyF?

La respuesta del bot debería contener https://www.luzyfuerza.org.ar/mi-mutualyf/ renderizada como link clickable (subrayado).

3. Verificaciones

  • El link aparece subrayado dentro del mensaje del bot
  • Tap/click → abre en nueva pestaña
  • Funciona en móvil (tap nativo)
  • Si el bot escribe https://x.com. (con punto al final), el punto NO entra al link
